# codedit-ui-js

vscode나 visual studio 같은 유용한 코드 에디터에서 제공하는 코드 윈도우 분할 기능, 크기조절기능, 위치 변경 기능 등의 유용한 기능을 웹에서 사용할 수 있도록 프레임워크 없이 순수 자바스크립트로 구현해보는 토이 프로젝트입니다.
